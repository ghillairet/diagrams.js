Diagram

diagram.extend(properties)

To create a Diagram with specific properties, you extend Diagram.Diagram and provide instance properties. 

                    
var ClassDiagram = Diagram.Diagram.extend({
    child: [
        'ClassShape'
    ]
});

var diagram = new ClassDiagram('canvas', 800, 600);

diagram.paper()

Returns the Raphael instance associated to this diagram.

                    
diagram.paper();

diagram.render()

Renders the diagram and all its elements.

                    
diagram.render();

diagram.el()

Returns the html container element for this diagram.

                    
diagram.el();

diagram.canvas()

Returns the SVG Element containing this diagram.

                    
diagram.canvas();

diagram.remove()

Removes the diagram and it's content from the canvas.

                    
diagram.remove();

diagram.zoom()

Performs a zoom.

                    
diagram.zoom();

diagram.createShape(function, attributes)

Creates a shape from a given function of type Diagram.Shape.

                    
var ClassShape = Diagram.Shape.extend({
    figure: {
        ...
    }
});

var shape = diagram.createShape(ClassShape, {x: 120, y: 20});
shape.render();

diagram.removeShape(shape)

Removes a Shape.

                    
diagram.removeShape(shape);

diagram.getShape(id)

Returns a Shape specified by it's id.

                    
diagram.getShape(shape.get('id'));
